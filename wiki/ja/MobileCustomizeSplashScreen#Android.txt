<div id='wikitext'>
<h1>Customizing the Splash screen of Your Mobile Application</h1>
<a class='iosRef' id='iosRef' href="javascript:ShowHideDiv('iosRef','specific-ios', 'iOS');"><img src="./uploads/Main/PlatformArrowDown.png"><h1>iOS</h1></a>
<div class='specific-ios' style='display: block;' id='specific-ios' > 
<p>Under iOS Basic, a default splash screen will be displayed while your game loads, oriented according to the <span class='doc-prop'>Default Screen Orientation</span> option in the <a class='wikilink' href='http://docwiki.unity3d.co.jp/index.php?n=Main.class-PlayerSettings'>Player Settings</a>.
</p>
<p class='vspace'>Users with an iOS Pro license can use any texture in the project as a splash screen. The size of the texture depends on the target device (320x480 pixels for 1-3rd gen devices, 1024x768 for iPad, 640x960 for 4th gen devices) and supplied textures will be scaled to fit if necessary. You can set the splash screen textures using the <a class='wikilink' href='http://docwiki.unity3d.co.jp/index.php?n=Main.class-PlayerSettings'>iOS Player Settings</a>.
</p>
<div class='vspace'></div></div>
<p class='vspace'><a name='Android' id='Android'></a>
</p><a class='androidRef' id='androidRef' href="javascript:ShowHideDiv('androidRef','specific-android', 'Android');"><img src="./uploads/Main/PlatformArrowDown.png"><h1>Android</h1></a>
<div class='specific-android' style='display: block;' id='specific-android' > 
<p>Under Android Basic, a default splash screen will be displayed while your game loads, oriented according to the <span class='doc-prop'>Default Screen Orientation</span> option in the <a class='wikilink' href='http://docwiki.unity3d.co.jp/index.php?n=Main.class-PlayerSettings'>Player Settings</a>.
</p>
<p class='vspace'>Android Pro users can use any texture in the project as a splash screen. You can set the texture from the Splash Image section of the Android <a class='wikilink' href='http://docwiki.unity3d.co.jp/index.php?n=Main.class-PlayerSettings'>Player Settings</a>. You should also select the <span class='doc-prop'>Splash scaling</span> method from the following options:-
</p><ul><li><span class='doc-prop'>Center (only scale down)</span> will draw your image at its natural size unless it is too large, in which case it will be scaled down to fit.
</li><li><span class='doc-prop'>Scale to fit (letter-boxed)</span> will draw your image so that the longer dimension fits the screen size exactly. Empty space around the sides in the shorter dimension will be filled in black.
</li><li><span class='doc-prop'>Scale to fill (cropped)</span> will scale your image so that the shorter dimension fits the screen size exactly. The image will be cropped in the longer dimension.
</li></ul><div class='vspace'></div></div>
